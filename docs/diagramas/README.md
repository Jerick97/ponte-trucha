## Iconos AWS

Los SVG de `aws-icons/` provienen del paquete oficial **AWS Architecture Icons,
release 30 de abril de 2026**:

- [AWS Architecture Icons](https://aws.amazon.com/architecture/icons/)
- Paquete usado:
  `Icon-package_04302026.4705b90f5aa45b019271a2699e9ce9b97b941ee1.zip`

AWS actualiza los iconos periódicamente. Revisar la página oficial antes del
pitch final y mantener los nombres/colores del paquete sin alterarlos.

Sentry y Mixpanel se muestran como cajas tipográficas, no como logos oficiales.
